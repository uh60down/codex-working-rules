# codex-working-rules

Shared Codex working rules, repo instructions, and reusable skills.

## Contents

- `AGENTS.md` - repository-level Codex instructions.
- `.agents/skills/bug-triage/SKILL.md` - evidence-first bug diagnosis workflow.
- `.agents/skills/domain-ontology/SKILL.md` - feature-domain mapping workflow.
- `.agents/skills/fact-check/SKILL.md` - verification gate for analytical write-ups.

## Using This Repo With Codex

Open this repository in Codex or Codex web. Codex reads `AGENTS.md` from the
repository root and discovers repo skills from `.agents/skills`.

For Codex web, connect GitHub at:

```text
https://chatgpt.com/codex
```

Then select `uh60down/codex-working-rules`.
